Introduction

This guide aims to help you build a three page website using valid HTML and CSS in order to prepare you for the Level 1 and Level 2 Achievement Standards 91073 and 91370. It doesn't mater which software you use to write your code in but Notepad++ and Brackets are good options, and both free. If you have a mac, look at TextWrangler.

I've chosen to make a site about the Camino de Santiago - because it is something I am interested in and because I have a lot of resources I can put into it without worrying about copyright. You can make your site about any topic you want, or feel free to use pictures and text from this guide.

How to use this guide

Read the explanations and copy the code into your own files. There will be suggestions of bits to tinker with which should help your understanding of how the different parts of the code work together. You should also look to add in your own code to test your understanding.

Let's get started

Web pages are simply text files. These files use a language called HTML to lay elements out on a page. HTML is written with tags that look like words enclosed in angle brackets. Before you can put anything on the page you need to get the basic structure down. Here is some code for a very simple webpage:

<!DOCTYPE html>
<html>
    <head>
        <title>Camino de Santiago</title>
    </head>
    <body>
        Hello World!
    </body>
</html>
