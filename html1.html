Let's get started

Web pages are simply text files. These files use a language called HTML to lay elements out on a page. HTML is written with tags that look like words enclosed in angle brackets. Before you can put anything on the page you need to get the basic structure down. Here is some code for a very simple webpage:

<!DOCTYPE html>
<html>
    <head>
        <title>Camino de Santiago</title>
    </head>
    <body>
        Hello World!
    </body>
</html>

Follow these steps to view your first web page (see screenshots below as well):

  1. Copy the code into your text editor.
  2. Save the file as 'index.html' into a new folder. Name the folder whatever your site is named.
  3. Open your browser chrome, firefox, safari etc and press Ctrl-o then find your file and open it.
