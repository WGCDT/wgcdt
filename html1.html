Let's get started

Web pages are simply text files. These files use a language called HTML to lay elements out on a page. HTML is written with tags which are words enclosed in angle brackets. Before you can put anything on the page you need to get the basic structure down. Here is some code for a very simple webpage:

<!DOCTYPE html>
<html>
    <head>
        <title>Camino de Santiago</title>
    </head>
    <body>
        Hello World!
    </body>
</html>

Follow these steps to view your first web page (see screenshots below as well):

  1. Copy the code into your text editor.
  2. Save the file as 'index.html' into a new folder. Name the folder whatever your site is named.
  3. Open your browser chrome, firefox, safari etc and press Ctrl-o then find your file and open it.

Every webpage should have these tag, but what are they?

  • <!DOCTYPE html> - This goes at the top of every web page document and tells the web browser which version of HTML the page is coded in.
  • <head> - Information about the page. You'll put some more stuff in here soon, but for now you should just have a title tag (see below).
  • <body> - This is the stuff you actually see on the page. You'll focus on this in the next section.
  • <title> - The title of the page, the browser will show this on the tab above the webpage when open.

You may have noticed that the tags are all in pairs - that's how you show the start and end of an element. For example with <h1>Camino de Santiago</h1>

the </h1> shows the end of the heading.

Putting text on your webpage

There is two main kinds of text in HTML - headings and paragraphs. Heading tags use h followed by a number (from 1 to 6) so <h1> or <h6>. Parapgraphs use <p> tags. Read the code snippet below and update your index.html using some heading and paragraph tags.

<!DOCTYPE html>
<html>
    <head>
        <title>Camino de Santiago</title>
    </head>
    <body>
        <h1>Camino de Santiago</h1>
        <p>The Camino is an ancient pilgrimage which either goes to Santiago de Compostela or through that city to Finisterre on the west coast of Spain. There are many  routes that one can follow on their <em>Camino</em> although the main routes follow Roman roads and trade routes. Traditionally a camino starts at the pilgrim's home but mdern day pilgrims head to significant places such as Saint-Jean-Pied-de-Port in France or Roncesvalles in  Northern Spain.</p>
        <p>Upon reaching Santiago de Compostela a pilgrim is eligible for a <em>compostela</em> if they have walked at least 100km cumulatively during their pilgrimage although it is not unusual to have walked more than 700km over 30 or 40 days. The most common route, the 'Camino Frances' or 'French Way' is approximately 769km alhough this can vary if the pilgrim takes detours or shortcuts.</p>
        <h2>The Credencial</h2>
        <p>The 'credencial' or 'passport' is how a pilgrim proves how far they have walked each day. The credencial is stamped at each 'albergue' (special camino hostel) and/or at a cafe along the route. Generally a pilgrim aims to get at least 2 stamps per day and some albergues require evidence that the pilgrim has walked at least 30km that day in order to get a bed.</p>
    </body>
</html>

Let's look at the new tags used here:

  • <h1> and <h2> - These give you headings. The smaller the number, the smaller the heading. You should try to use these headings correctly (h1 for the main headings, h2 forsubheadings, etc) as they have other meaning.
  • <p> - Paragraphs. The important thing is to use paragraph tags for each and every paragraph
  • <em> - Emphasised text, this really means italics. There are other ways to do this but if you want to make a word stand out easily, use em. Try replacing a pair of <em> tags with some <strong> tags.
